# doanpython
# 🐍 Snake Game

Game Rắn Săn Mồi được lập trình bằng Python và thư viện Pygame.

## 🎮 Tính năng

- **Menu chính** — Play, Option, Quit
- **Gameplay** — điều khiển rắn ăn mồi, tăng điểm
- **Mồi đặc biệt** — cứ mỗi 15 giây, mồi phình to và cho **+10 điểm**
- **Game Over** — hiển thị điểm, chọn chơi lại hoặc về menu
- **Option** — chỉnh âm lượng và độ sáng
- **Icon con rắn** — hiển thị trên cửa sổ và taskbar

## 🕹️ Điều khiển

| Phím | Chức năng |
|------|-----------|
| ↑ ↓ ← → hoặc WASD | Di chuyển rắn |
| P hoặc SPACE | Tạm dừng / Tiếp tục |
| ESC | Về menu |


## 🛠️ Cài đặt và chạy

**Yêu cầu:** Python 3.8 trở lên

**1. Clone repo về máy:**
git clone https://github.com/pvelyse247/doanpython.git
cd doanpython


**2. Cài thư viện:**
pip install pygame

**3. Chạy game:**
python snake_game_final.py
